O problema real

Apps de produtividade foram feitos para outros cérebros

Para quem tem TDAH ou TEA, ver uma lista enorme de tarefas atrasadas não gera ação. Gera paralisia.

  • Ver 87 tarefas atrasadas congela o cérebro — você fecha o app sem fazer nada

Design neuro-inclusivo

Construído com 7 princípios que respeitam seu cérebro

Cada detalhe foi pensado para reduzir atrito e aumentar o que importa: fazer uma coisa por vez, sem se sentir pequeno.

👁
Revelação progressiva
A informação aparece quando você precisa, não toda de uma vez. Cada tela mostra só o essencial para o momento atual.
🔁
Padrões previsíveis
Consistência é segurança. Todo elemento se comporta da mesma forma sempre — sem surpresas que quebram o fluxo mental.
🧠
Carga cognitiva zero
Menos decisões desnecessárias. Defaults inteligentes para quem não quer configurar tudo — mas pode personalizar o que quiser.
🌿
Tom nunca punitivo
Tarefas não feitas não ficam vermelhas. Não há contador de atraso. O app torce por você — nunca te julga. Âmbar suave, nunca vermelho.
