Os 3 conceitos

Três formas de turbinar o Claude

Cada conceito resolve um problema diferente. Entender os três muda como você usa o ecossistema.

Skill
Um arquivo de instruções que o Claude carrega automaticamente quando você usa uma palavra-chave. É como ligar um modo especialista — ele já sabe o contexto, o produto, a persona, o tom de voz.
Ativa por keyword Contexto embutido Um especialista
Agente
Uma skill com comportamento mais autônomo. Em vez de você pedir formato por formato, o agente analisa o objetivo e entrega um pacote completo — vários outputs de uma vez só.
Entrega batch Escolhe o formato Mais autônomo
Xquad
Um squad de múltiplos especialistas que trabalham juntos. O Claude assume vários personagens ao mesmo tempo — cada um com seu framework, perspectiva e especialidade.
Múltiplos personagens Perspectivas diversas Decisão em grupo
Skill na prática

Como uma skill funciona

Você digita uma keyword. O Claude detecta, carrega o SKILL.md, e já começa com todo o contexto pronto — sem você precisar explicar nada.

Você digita no terminal: "cria carrossel tractorpro"
Claude detecta a keyword criativo tractorpro
Carrega o SKILL.md com persona, produto e formatos
Entrega o carrossel completo, 10 slides, pronto
Terminal Claude Code — skill criativo-tractorpro ativa

Os Xquads

Squads de especialistas trabalhando juntos

Cada Xquad é um time. Você faz uma pergunta — vários especialistas respondem com seus respectivos frameworks. É como ter um board de consultores disponível 24h.

Exemplo — Hormozi Squad
M
Max
hormozi squad — preciso criar uma oferta pro TractorPRO, R$97/mes, gestor de tráfego
💰
Hormozi Squad — 16 agentes ativos
Offer Architect · Pricing Strategist · Lead Engine · 13 outros
Offer Architect: R$97 é o ponto certo. Menos de 1 dia de trabalho de 1 cliente. A oferta precisa de um value stack claro — liste o que eles ganham em dinheiro/tempo por mês.

Pricing Strategist: Âncora no custo do problema — 3h/semana de relatório × R$150/h = R$450/semana de custo. Você está cobrando R$97/mês para economizar R$1.800/mês. Essa é a headline.

Usando na prática

Do zero ao resultado em 3 passos

Tudo começa no terminal. Ative a skill, abra o Claude, e use a keyword.

Usando uma Skill

1
Ative a skill
Na página Skills, ligue o toggle da skill que quer usar. Ela é instalada no Claude automaticamente.
2
Abra o terminal
Acesse o terminal web ou rode claude no VPS.
3
Use a keyword
Digite a keyword e o Claude já sabe o contexto completo.
cria carrossel tractorpro sobre relatório
copy squad — headline pro TractorPRO
hormozi — oferta R$97/mês gestor

Usando um Xquad

1
Ative o squad
Na página Skills, role até o grupo Xquads e ligue o squad que quer usar.
2
Chame o squad + contexto
Diga o nome do squad e o seu desafio em uma frase. Quanto mais contexto, melhor a resposta.
advisory board — devo lançar TractorPRO
em modo B2B ou B2C primeiro?

traffic masters — campanha Meta Ads
gestor de tráfego, CPL meta R$8
3
Receba múltiplas perspectivas
Cada especialista responde com seu framework. Você decide qual perspectiva aplicar.
