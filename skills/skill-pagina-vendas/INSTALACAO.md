## O que e esta Skill?

A **PageCraft** e uma skill do Claude Code que gera paginas de vendas COMPLETAS e profissionais do zero:
- Copy estrategica usando 8 frameworks (AIDA, PASTOR, 4Ps, FAB, ACCA, SSS, QUEST, 1-2-3-4)

- Mobile-first, responsivo, com elementos de conversao
- Meta Pixel, GTM, WhatsApp, countdown, sticky CTA, FAQ accordion

## Como Usar

### Opcao 1 — Interativo (Recomendado para primeira vez)

Diga ao Claude:
```
Crie uma pagina de vendas para o meu [produto/servico]
```

O Claude vai te fazer 8 perguntas estrategicas (uma por vez):
1. Qual e seu produto/servico?
2. Quem e seu cliente ideal?
3. Nivel de consciencia do publico (frio/morno/quente)
4. Que provas voce tem?
5. Voce tem metodo/historia?
6. Tom de voz
7. Referencia visual
8. Descricao para fotos IA

Depois ele gera TUDO automaticamente.

### Opcao 2 — Direto (Se ja tem tudo pronto)

Cole tudo de uma vez:
```
Crie pagina de vendas:
- Produto: [nome e descricao]
- Preco: R$ [valor]
- Publico: [quem e]
- Nivel: [frio/morno/quente]
- Provas: [depoimentos, numeros, etc]
- Tom: [autoritario/empatico/tecnico/energetico/intimo]
- Historia: [resumo da sua historia]

Gerar: copy completa + HTML production-ready
```

## O que a Skill Entrega

### Fase 1 — Briefing Inteligente
- Diagnostico do seu produto/publico
- Selecao automatica do melhor framework

### Fase 2 — Estrategia
- Framework principal + complementares
- Mapa completo das secoes da pagina
- Planejamento das fotos IA

### Fase 3 — Copy Completa
- **3 variacoes de Hero** (emocional, racional, curiosidade)
- Pontuacao de cada variacao
- Todas as secoes com notas estrategicas
- Stack de valor com ancoragem
- Garantia estruturada
- FAQ que quebra objecoes
- CTA final + P.S.

### Fase 4 — Analise
- Score geral da copy (1-10)
- Mapa de emocoes por secao
- Objecoes cobertas vs descobertas
- Sugestoes de melhoria

### Fase 5 — Design + HTML
- Pagina HTML production-ready
- Mobile-first responsivo
- Tailwind CSS
- Countdown timer
- WhatsApp flutuante
- Sticky CTA mobile
- FAQ accordion
- Placeholders para Meta Pixel e GTM
- Pronta para publicar!

## Frameworks Disponiveis

| Framework | Melhor Para |
|-----------|-------------|
| **AIDA** | Uso geral, paginas rapidas, qualquer ticket |
| **PASTOR** | High ticket, mentorias, transformacao complexa |
| **4Ps** | Lancamentos, urgencia, publico quente |
| **FAB** | SaaS, B2B, produtos com muitas features |
| **ACCA** | Publico cetico, precisa de prova pesada |
| **SSS** | Marca pessoal, historia forte, publico frio |
| **QUEST** | Educacional, publico analitico, solucoes complexas |
| **1-2-3-4** | Low ticket, e-commerce, paginas rapidas |

A skill escolhe automaticamente o melhor framework baseado no seu produto, publico e ticket.

---

## Dicas Importantes

1. **Quanto mais contexto voce der, melhor a copy** — nao economize nos detalhes
2. **Tenha sua persona pronta** — se nao tiver, a skill cria uma pra voce
3. **Provas sao OURO** — depoimentos, numeros e cases fazem a copy disparar
4. **Historia vende** — se voce tem uma historia de metodo, use
5. **Teste as 3 variacoes de hero** — troque a cada 2 semanas e veja qual converte mais
6. **Sempre instale pixels** — Meta Pixel e GTM sao obrigatorios
7. **Mobile primeiro** — 70%+ do seu trafego e mobile
