# QUEST Framework (Qualify, Understand, Educate, Stimulate, Transition)

<!-- SYSTEM: QUEST is a consultative-selling framework. It filters the right audience first (Qualify), shows deep empathy (Understand), educates on the real problem (Educate), paints the desired future (Stimulate), then smoothly transitions to the offer (Transition). Never skip Qualify — it increases conversion by making the reader feel "this is for ME." -->

---

## 1. QUALIFY (Qualificar)

### 6 Formulas

#### 1.1 Direct Qualification
```
Isso é para você se: (critério 1), (critério 2) e (critério 3).
```
```
Se você é (perfil) que quer (resultado) mas ainda (obstáculo), continue lendo.
```

#### 1.2 Qualification by Questions
```
Responda mentalmente:
→ Você (pergunta sobre situação atual)?
→ Já tentou (solução comum) sem resultado?
→ Sente que (emoção/frustração)?

Se respondeu "sim" pra pelo menos duas, essa página foi feita pra você.
```

#### 1.3 Qualification by Scenario
```
Se você está nesse cenário agora…

→ (Cenário detalhado 1 — dia a dia do público)
→ (Cenário detalhado 2 — tentativa frustrada)
→ (Cenário detalhado 3 — emoção recorrente)

…então o que eu vou te mostrar aqui pode mudar tudo.
```

#### 1.4 Exclusion-First
```
Antes de tudo, vou ser honesto: isso NÃO é para todo mundo.

❌ Não é pra quem (perfil excluído 1 — ex: "quer resultado sem esforço")
❌ Não é pra quem (perfil excluído 2 — ex: "não está disposto a investir em si mesmo")
❌ Não é pra quem (perfil excluído 3)

✅ É pra quem (perfil qualificado — descrito positivamente).
```

#### 1.5 Qualification by Type/Profile
```
Eu trabalho com (número) tipos de (público):

**Tipo 1: O (nome do perfil)** — (descrição em 1 frase). Se é você, (o que vai encontrar aqui).
**Tipo 2: O (nome do perfil)** — (descrição). Se é você, (o que vai encontrar).
**Tipo 3: O (nome do perfil)** — (descrição). Se é você, (o que vai encontrar).

Independente do tipo, o (método/produto) funciona porque (razão universal).
```

#### 1.6 Self-Selection
```
Essa página é para (perfil específico) que está em um desses momentos:

1. (Momento de vida/carreira 1 — ex: "Começando do zero e sem saber por onde ir")
2. (Momento 2 — ex: "Já tem experiência mas está estagnado")
3. (Momento 3 — ex: "Faturando bem mas sem estrutura para escalar")

Se você se vê em qualquer um desses momentos, continue.
```

### Qualify Templates

#### Complete Qualification
```
## Esta página é para você?

Eu sei que seu tempo é valioso, então vou ser direto.

O (produto) foi criado especificamente para (perfil detalhado) que:

✅ (Critério 1 — situação atual)
✅ (Critério 2 — desejo/objetivo)
✅ (Critério 3 — disposição/mentalidade)

Se isso soa como você, cada palavra nessa página foi escrita pensando na sua situação.

Se não, tudo bem — (frase respeitosa de desqualificação).
```

#### Qualification by Questions
```
Antes de continuar, me responda:

1. Você (pergunta sobre dor/situação)?
2. Já tentou (solução) e não funcionou como esperava?
3. Acredita que (crença positiva sobre possibilidade)?
4. Está disposto a (compromisso necessário)?

Se respondeu sim pra (2 ou mais/todas), continue lendo — o que vem a seguir foi feito sob medida pra você.
```

## 2. UNDERSTAND (Compreender)

### 6 Formulas

#### 2.1 Deep Comprehension
```
Eu sei o que é (situação do público). Eu sei como é (sentimento específico) quando (cenário detalhado).
```
```
Você acorda de (manhã/madrugada) pensando em (preocupação). Passa o dia inteiro tentando (esforço). E no final, sente que (frustração).
```

#### 2.2 Daily Routine
```
Seu dia provavelmente é assim:

(Manhã) — (rotina com dor sutil)
(Tarde) — (esforço que não dá resultado)
(Noite) — (consequência emocional)

E amanhã? Tudo de novo.
```

#### 2.3 Frustrated Attempts
```
Você já tentou de tudo:

→ (Tentativa 1) — e (por que não funcionou)
→ (Tentativa 2) — e (resultado frustrante)
→ (Tentativa 3) — e (o que sentiu depois)

E agora está se perguntando: "Será que o problema sou eu?"

Não. O problema nunca foi você. Foi (revelação da causa real).
```

#### 2.4 Emotional Comprehension
```
Eu sei que por trás de (objetivo declarado — ex: "querer ganhar mais dinheiro"), existe algo maior:

→ (Desejo profundo 1 — ex: "a liberdade de não depender de ninguém")
→ (Desejo profundo 2 — ex: "o orgulho de mostrar pra quem duvidou")
→ (Desejo profundo 3 — ex: "a segurança de dar o melhor pros seus filhos")

Eu entendo. E é exatamente por isso que criei o (produto).
```

#### 2.5 Deep Desire
```
No fundo, o que você realmente quer não é (resultado superficial).

O que você quer é (desejo emocional profundo).

Quer (cena visual do desejo — ex: "abrir o aplicativo do banco e sorrir"). Quer (cena 2 — ex: "olhar no espelho e reconhecer alguém que admira"). Quer (cena 3 — ex: "dormir tranquilo sabendo que amanhã está resolvido").
```

#### 2.6 Mirror Statement
```
Se eu pudesse adivinhar, diria que agora você está pensando:

"(Pensamento interno 1 — dúvida)"
"(Pensamento interno 2 — medo)"
"(Pensamento interno 3 — esperança cautelosa)"

Acertei? Então continue lendo.
```

### Understand Templates

#### Deep Comprehension Template
```
Eu preciso que você saiba de uma coisa: eu te entendo.

Não é empatia de fachada. Eu sei o que é (situação específica vivida pelo criador ou muito estudada).

Sei como é (emoção 1) quando (cenário). Sei como é (emoção 2) quando (cenário). E sei como é aquela voz na cabeça dizendo "(pensamento autodepreciativo do público)".

Eu sei porque (razão — experiência pessoal / anos trabalhando com esse público / centenas de conversas).

E é por isso que o que eu vou te mostrar agora é diferente de tudo que você já viu.
```

## 3. EDUCATE (Educar)

### 6 Formulas

#### 3.1 Root Cause Revelation
```
O verdadeiro motivo pelo qual (problema) acontece não é (o que a maioria pensa). É (causa raiz revelada).

(Explicação de por que a causa raiz é contra-intuitiva).

Quando você resolve (causa raiz), (resultado 1), (resultado 2) e (resultado 3) acontecem naturalmente.
```

#### 3.2 Myth-Breaking
```
Mito: "(crença popular que o público tem)"
Realidade: (verdade com evidência)

Esse mito é perigoso porque (consequência de seguir o mito). A verdade é que (abordagem correta).
```

#### 3.3 Teaching a Framework
```
Eu chamo isso de (nome do método/framework).

Funciona em (número) pilares:

**Pilar 1: (Nome)** — (explicação em 1-2 frases + por que importa)
**Pilar 2: (Nome)** — (explicação + por que importa)
**Pilar 3: (Nome)** — (explicação + por que importa)

Quando esses (número) pilares trabalham juntos, (resultado transformador).
```

#### 3.4 Successful Pattern
```
Analisei (número) (casos/clientes/empresas) que (resultado positivo). Todos tinham (número) coisas em comum:

1. (Padrão 1)
2. (Padrão 2)
3. (Padrão 3)

E os que NÃO tiveram resultado? Faltava (exatamente esses elementos).
```

#### 3.5 Analogy Education
```
Pense em (tema) como (analogia familiar).

(Ação errada) é como (analogia negativa — ex: "construir uma casa sem fundação").
(Ação certa) é como (analogia positiva — ex: "primeiro a fundação, depois as paredes, depois o telhado").

O (produto) é o equivalente a (elemento da analogia que resolve tudo).
```

#### 3.6 Contrast Education
```
Existem duas abordagens para (resultado):

**A abordagem comum:** (O que a maioria faz → por que não funciona → consequência)

**A abordagem certa:** (O que funciona → por que funciona → resultado)

A diferença? (Elemento-chave que separa as duas).
```

### Educate Templates

#### Root Cause Revelation Template
```
## Por que (problema) realmente acontece

Todo mundo acha que o problema é (causa superficial). "É só (solução óbvia)", dizem.

Mas se fosse tão simples, por que (número)% das pessoas ainda (sofre com o problema)?

A verdade é que o problema não está em (onde a maioria procura). Está em (localização real da causa raiz).

(Explicação detalhada do mecanismo — como a causa raiz gera o problema).

É por isso que (soluções comuns) não funcionam: elas tratam o (sintoma), não a (causa).

O (método/produto) foi construído para resolver exatamente isso.
```

#### Myth-Breaking Template
```
## As (número) mentiras que te contaram sobre (tema)

**Mentira #1: "(mito popular)"**
A verdade: (realidade com dado/evidência). (Por que esse mito persiste).

**Mentira #2: "(mito popular)"**
A verdade: (realidade). (Consequência de acreditar nesse mito).

**Mentira #3: "(mito popular)"**
A verdade: (realidade). (O que muda quando você descarta esse mito).

Quando você para de acreditar nessas mentiras, (resultado positivo) se torna inevitável.
```

## 4. STIMULATE (Estimular)

### 5 Formulas

#### 4.1 Future Visualization
```
Imagine daqui a (prazo):

Você acorda (cena positiva matinal). Olha pro (celular/computador/espelho) e vê (indicador de sucesso). Sente (emoção positiva) porque sabe que (conquista consolidada).

Isso não é fantasia. É o que acontece quando (condição — usar o método/seguir o caminho).
```

#### 4.2 Future Contrast
```
Daqui a (prazo), você vai estar em um de dois lugares:

**Cenário A (sem ação):** (Descrição detalhada da vida estagnada — mesma rotina, mesmas frustrações, mesmos resultados)

**Cenário B (com ação):** (Descrição detalhada da vida transformada — novos resultados, novas emoções, nova identidade)

A única coisa que separa esses dois cenários é (a decisão que você toma hoje).
```

#### 4.3 Possibilities
```
Com (método/resultado), as possibilidades se abrem:

→ Você pode (possibilidade 1 — profissional)
→ Você pode (possibilidade 2 — financeira)
→ Você pode (possibilidade 3 — pessoal/emocional)
→ Você pode até (possibilidade 4 — aspiracional/surpreendente)
```

#### 4.4 Stimulating Questions
```
O que você faria se (resultado financeiro)? Para onde viajaria se (liberdade de tempo)? Como seria seu (dia/vida/relacionamento) se (transformação emocional)?

Não são perguntas retóricas. São possibilidades reais para quem (condição).
```

#### 4.5 Identity Stimulation
```
Você não vai ser mais "aquela pessoa que (identidade negativa atual)."

Vai ser reconhecido como (identidade aspiracional). As pessoas vão dizer "(frase que os outros dirão sobre você)."

E o mais importante: você vai se olhar no espelho e ver (versão ideal de si mesmo).
```

### Stimulate Templates

#### Future Visualization Template
```
## Fecha os olhos por um segundo e imagina isso...

É (dia da semana), (horário). Você (ação da nova rotina — ex: "abre o notebook no café favorito").

(Detalhe sensorial 1 — o que vê). (Detalhe sensorial 2 — o que sente). (Detalhe sensorial 3 — o que pensa).

Seu (indicador de sucesso — ex: "faturamento do mês") é (número aspiracional). Seus (clientes/alunos/seguidores) (ação positiva — ex: "enviam mensagens de gratidão").

Você sorri. Não por causa do (dinheiro/número). Mas porque finalmente sente (emoção profunda — ex: "que está no controle da própria vida").

Isso é real. É exatamente o que (número) (alunos/clientes) vivem hoje.
```

## 5. TRANSITION (Transição)

### 5 Formulas

#### 5.1 Smooth Transition
```
E é exatamente pra isso que existe o (produto).
```
```
Tudo isso que acabei de descrever? É o que o (produto) entrega. Deixa eu te mostrar como.
```

#### 5.2 Revelation Transition
```
O (método/framework) que torna isso possível tem um nome: (nome do produto).

Eu passei (tempo) desenvolvendo, testando com (número) (alunos/clientes), refinando cada detalhe. E agora está pronto pra você.
```

#### 5.3 Invitation Transition
```
Se tudo que você leu até aqui fez sentido, eu quero te fazer um convite.

Não é um convite qualquer. É um convite pra (transformação) — com (suporte/método/acompanhamento) do início ao fim.
```

#### 5.4 "That's Why" Transition
```
Foi por causa de tudo isso — a (dor que entendo), os (mitos que quebrei), o (futuro que é possível) — que eu criei o (produto).

Não é mais um (curso/mentoria/programa). É (descrição diferenciada do produto).
```

#### 5.5 Transition + Offer Presentation
```
Apresento o **(nome do produto)**: (descrição em 1 frase — o que é, pra quem, qual resultado).

Dentro dele, você encontra:

📦 (Componente 1) — (benefício)
📦 (Componente 2) — (benefício)
📦 (Componente 3) — (benefício)

Tudo o que você precisa para sair de (ponto A) e chegar em (ponto B).
```

### Transition Templates

#### Complete Transition
```
Agora que você entende (resumo do Educate — a causa real), e consegue visualizar (resumo do Stimulate — o futuro possível)...

A pergunta é: como chegar lá?

A resposta é o **(nome do produto)**.

(Descrição em 2-3 frases do que é, como funciona, e por que é diferente).

Veja exatamente o que você recebe:

(Lista detalhada de componentes com benefícios)
```

#### Invitation Transition
```
Se você chegou até aqui, já sabe que (insight principal do Educate).

Já consegue imaginar (visão do Stimulate).

Agora, eu quero te convidar a dar o próximo passo.

Não é sobre comprar um (produto). É sobre decidir que (transformação desejada) é prioridade pra você.

E quando você decide isso, o (produto) é o caminho mais rápido e seguro que eu conheço.
```

## WHEN TO USE QUEST

<!-- SYSTEM: QUEST excels when you need to filter your audience AND educate them. The Qualify step increases relevance and the Understand step builds trust through demonstrated empathy. Best for offers where the audience doesn't yet know the real solution. -->

**Best for:**
- Ofertas que precisam filtrar o público certo
- High-ticket onde empatia é essencial antes da venda
- Mercados onde o público precisa ser educado sobre a causa real
- Mentorias, consultorias e programas de transformação
- Páginas de vendas longas com storytelling educativo
- Quando a concorrência é genérica e você precisa se diferenciar pela profundidade

**Not ideal for:**
- Low-ticket de impulso (qualificação demais pode afastar)
- Ofertas auto-explicativas
- Quando o público já está decidido a comprar

## QUEST vs ACCA vs PASTOR

| Aspecto | QUEST | ACCA | PASTOR |
|---------|-------|------|--------|
| Primeiro passo | Qualificar (filtrar) | Conscientizar | Identificar a Dor |
| Motor principal | Empatia + Educação | Lógica + Prova | Dor → Alívio |
| Público ideal | Precisa ser filtrado | Sofisticado/cético | Consciente da dor |
| Diferencial | Qualificação + Empatia profunda | Compreensão lógica | Agitação emocional |
| Ticket | Médio-alto | Médio-alto | Qualquer |
| Comprimento | Longo (5 etapas detalhadas) | Médio-longo | Médio |
| Melhor para | Mentorias, consultorias | SaaS, produtos técnicos | Qualquer oferta |
| Risco | Qualificar demais e perder público | Educar demais e entediar | Exagerar na dor |
