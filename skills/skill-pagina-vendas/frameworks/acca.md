# ACCA Framework (Awareness, Comprehension, Conviction, Action)

<!-- SYSTEM: ACCA is an educational-persuasion framework. Unlike AIDA which hooks emotionally, ACCA builds understanding first, then conviction through logic and proof. Best for sophisticated audiences and complex offers. Progress A→C→C→A sequentially — never skip Comprehension. -->

---

## 1. AWARENESS (Consciência)

### 6 Formulas

#### 1.1 Revealing Question
```
Você sabia que (estatística surpreendente) dos (público) (resultado negativo) por causa de (causa oculta)?
```
```
Já parou pra pensar por que (situação frustrante acontece) mesmo quando você (esforço que a pessoa faz)?
```

#### 1.2 Shocking Stat
```
(Número impressionante) de (público) (ação/resultado negativo) todos os (período). E o pior: (detalhe agravante).
```
```
Em (ano), (pesquisa/fonte) revelou que (dado chocante). A pergunta é: você faz parte dessa estatística?
```

#### 1.3 Myth-Breaking
```
Você foi ensinado que (crença popular). Mas a verdade é exatamente o oposto: (realidade).
```
```
A maior mentira que te contaram sobre (tema) é que (mito). Na verdade, (verdade contra-intuitiva).
```

#### 1.4 Hidden Cost
```
Cada (dia/mês/semana) que você passa (fazendo/sem fazer algo), você perde (custo real — dinheiro, tempo, oportunidade).
```
```
O preço de (não resolver problema) não é só (custo óbvio). É também (custo emocional) e (custo de oportunidade).
```

#### 1.5 Comparison
```
Enquanto (grupo A) continua (comportamento antigo), (grupo B) já descobriu que (abordagem nova) entrega (resultado superior).
```
```
Existem dois tipos de (profissional/pessoa): os que (abordagem errada) e os que (abordagem certa). A diferença? (Resultado contrastante).
```

#### 1.6 "What If"
```
E se eu te dissesse que existe uma forma de (resultado desejado) sem (sacrifício/dor esperada)?
```
```
E se o motivo pelo qual você ainda não (resultado) não fosse (causa que a pessoa acredita), mas sim (causa real)?
```

### Awareness Templates

#### Hidden Problem
```
(Público) enfrenta um problema que a maioria nem sabe que existe.

Todo (dia/mês), (ação comum aparentemente inofensiva) silenciosamente (consequência negativa). É como (analogia visual — ex: "um vazamento pequeno que, com o tempo, destrói a fundação inteira").

E o mais perigoso? (Por que a pessoa não percebe — ex: "os sintomas parecem normais").
```

#### Symptom vs Cause
```
O que você acha que é o problema: (sintoma visível — ex: "falta de clientes").
O que realmente é o problema: (causa raiz — ex: "posicionamento genérico que não diferencia você de ninguém").

Resolver o sintoma é como (analogia de solução superficial). Resolver a causa é (resultado real).
```

#### Reality Contrast
```
O que te disseram: "(conselho genérico/crença popular)"
O que funciona de verdade: "(abordagem contra-intuitiva)"

(Número)% dos (público) que (resultado positivo) fizeram exatamente o contrário do que o mercado ensina.
```

#### Numbers
```
(Número grande e específico).

Esse é o número de (métrica impactante — ex: "pessoas que desistiram de empreender nos últimos 12 meses").

O motivo? Não foi falta de (esforço/inteligência/vontade). Foi falta de (solução que você oferece).
```

#### Opportunity
```
Existe uma janela de oportunidade aberta agora em (mercado/nicho) que a maioria ainda não enxergou.

Enquanto (massa) continua brigando por (mercado saturado), quem entendeu (insight) está (resultado surpreendente).

Essa janela não vai ficar aberta para sempre.
```

## 2. COMPREHENSION (Compreensão)

### 5 Formulas

#### 2.1 Causal Explanation
```
Isso acontece porque (mecanismo — causa e efeito claro). Quando você (ação A), o resultado natural é (consequência B). É por isso que (conclusão lógica).
```

#### 2.2 Analogy
```
Pense assim: (situação do público) é como (analogia do cotidiano). Se você (ação na analogia), o resultado é (consequência na analogia). Da mesma forma, quando você (ação no contexto real), você (resultado real).
```

#### 2.3 Deconstruction
```
O problema tem (número) camadas:
1. (Camada superficial — o que todo mundo vê)
2. (Camada intermediária — o que poucos percebem)
3. (Camada profunda — a causa raiz que ninguém fala)

A maioria tenta resolver a camada 1. Quem resolve a camada 3 transforma tudo.
```

#### 2.4 Process
```
Funciona assim:
Passo 1: (Ação inicial) — isso (resultado intermediário 1)
Passo 2: (Ação seguinte) — isso (resultado intermediário 2)
Passo 3: (Ação final) — e aí (resultado final desejado)
```

#### 2.5 Elimination
```
Não é (causa errada 1). Não é (causa errada 2). E definitivamente não é (causa errada 3 — a mais popular).

O verdadeiro motivo pelo qual (problema persiste) é (causa real — revelação).
```

### Comprehension Templates

#### Root Cause
```
Por que (problema) acontece mesmo quando você (esforço)?

A resposta é mais simples do que parece: (explicação da causa raiz em 1-2 frases).

(Analogia que torna a causa raiz visual e tangível).

Quando você entende isso, (mudança de perspectiva que o leitor experimenta).
```

#### Why Others Fail
```
A maioria das pessoas que tenta (objetivo) segue esse caminho:

❌ (Abordagem errada 1) — que leva a (resultado frustrante)
❌ (Abordagem errada 2) — que causa (efeito colateral)
❌ (Abordagem errada 3) — que funciona no curto prazo mas (consequência)

O que funciona é diferente: (abordagem correta resumida).
```

#### Solution Mechanism
```
O (nome do método/framework) funciona porque ataca (causa raiz) diretamente.

Em vez de (abordagem comum), ele (mecanismo da solução).

É como (analogia): enquanto (método errado) é como (comparação negativa), o (seu método) é como (comparação positiva).

O resultado? (Consequência positiva mensurável).
```

#### Analogy Template
```
Imagine que (situação do público) é como (cenário familiar).

Se você (ação errada na analogia), o que acontece? (Resultado negativo na analogia).

Mas se você (ação certa na analogia), aí sim: (resultado positivo na analogia).

(Nome do produto/método) é o equivalente a (ação certa na analogia) para o seu (contexto real).
```

#### Visual Framework
```
Pense no (conceito) como um (objeto visual — pirâmide, escada, engrenagem):

🔹 Base: (fundamento — sem isso nada funciona)
🔹 Meio: (elemento que conecta/amplifica)
🔹 Topo: (resultado/conquista final)

A maioria pula direto pro topo. É por isso que (falha).
```

#### Q&A
```
"Mas (objeção/dúvida comum 1)?"
→ (Resposta clara e educativa)

"E se (preocupação 2)?"
→ (Resposta com prova/lógica)

"Isso funciona pra (caso específico 3)?"
→ (Resposta com exemplo/evidência)
```

## 3. CONVICTION (Convicção)

### 5 Formulas

#### 3.1 Results
```
(Nome do cliente/aluno) estava (situação antes). Em (prazo), depois de (ação com seu método), (resultado específico e mensurável).
```
```
(Número) (alunos/clientes/empresas) já (resultado) usando (método/produto). Aqui estão alguns deles: (lista de resultados).
```

#### 3.2 Logic
```
Se (premissa lógica A — algo que o leitor concorda), e (premissa lógica B — também inegável), então a única conclusão lógica é (sua proposta).
```
```
A matemática é simples: (cálculo que prova o valor). Mesmo que (cenário conservador), o retorno seria de (resultado).
```

#### 3.3 Risk Elimination
```
Se em (prazo) você não (resultado prometido), eu devolvo (100%/cada centavo) do seu investimento. Sem perguntas, sem burocracia.
```
```
O risco é 100% meu. Se não funcionar pra você, o prejuízo é meu — não seu.
```

#### 3.4 Cost Contrast
```
O custo de (não agir) é (valor alto — tempo perdido, dinheiro desperdiçado, oportunidade perdida). O investimento no (produto) é (valor — fração do custo de não agir).
```
```
Você pode continuar gastando (valor) por (mês/ano) em (soluções que não funcionam). Ou investir (valor menor) uma vez e (resultado definitivo).
```

#### 3.5 Authority
```
Nos últimos (período), eu (credencial — número de alunos, anos de experiência, resultados alcançados). Isso me deu uma visão que poucos têm: (insight exclusivo).
```
```
Depois de (credencial impressionante), eu condensei tudo em (produto) — para que você não precise passar pelo mesmo caminho.
```

### Conviction Templates

#### Layered Social Proof
```
Não acredite em mim. Acredite nos resultados.

📊 (Número) (alunos/clientes) em (período)
⭐ (Nota média/avaliação)
🏆 (Resultado mais impressionante de um cliente)

Veja o que eles dizem:

"(Depoimento 1 — resultado específico)" — (Nome, contexto)
"(Depoimento 2 — transformação emocional)" — (Nome, contexto)
"(Depoimento 3 — comparação antes/depois)" — (Nome, contexto)
```

#### Logic Proof
```
Vamos aos fatos:

→ (Dado/estatística que suporta premissa 1)
→ (Dado/estatística que suporta premissa 2)
→ (Conclusão lógica inevitável)

Não é opinião. É (matemática/ciência/dado de mercado).
```

#### Total Risk Elimination
```
Sua proteção é total:

🛡️ Garantia (tipo — incondicional/condicional) de (prazo) dias
📋 Se (condição — assistir as aulas/aplicar o método) e não (resultado), devolvemos (100%/cada centavo)
🤝 (Detalhe extra de segurança — suporte durante o período, acompanhamento, etc.)

O risco é ZERO pra você. Todo o risco é meu.
```

#### Authority Template
```
Quem criou isso?

(Nome) é (títulos/credenciais relevantes).

→ (Feito impressionante 1)
→ (Feito impressionante 2)
→ (Feito impressionante 3)

(Frase que conecta autoridade ao benefício do leitor — ex: "Tudo isso foi condensado neste método para que você tenha em semanas o que levou anos para construir.")
```

#### Cost of Inaction
```
Qual o preço de NÃO agir?

Se nos próximos (prazo), nada mudar, você vai continuar:
→ (Consequência negativa 1 — financeira)
→ (Consequência negativa 2 — emocional)
→ (Consequência negativa 3 — oportunidade perdida)

Some tudo isso. O custo de não decidir é (muito maior/infinitamente maior) que o investimento.
```

## 4. ACTION (Ação)

### 5 Formulas

#### 4.1 Direct CTA
```
Clique no botão abaixo e (ação imediata — garanta sua vaga/comece agora/acesse agora).
```

#### 4.2 Urgency CTA
```
(Condição de escassez — vagas limitadas/preço sobe em X horas/bônus expira). (CTA direto).
```

#### 4.3 Two Options
```
Você tem dois caminhos agora:

Caminho 1: (Continuar como está — consequência negativa).
Caminho 2: (Agir agora — resultado positivo).

A escolha é sua. Mas só um desses caminhos te leva a (resultado desejado).
```

#### 4.4 Emotional Close
```
Daqui a (prazo), você vai estar em um de dois lugares: (cenário positivo se agir) ou (cenário negativo se não agir). A diferença é a decisão que você toma agora.
```

#### 4.5 Recap + CTA
```
Recapitulando: você recebe (item 1) + (item 2) + (item 3), com (garantia), por apenas (preço). (CTA).
```

### Action Templates

#### Structured CTA
```
## Comece agora

Tudo o que você precisa para (resultado) está aqui.

✅ (Item 1 com valor percebido)
✅ (Item 2 com valor percebido)
✅ (Item 3 com valor percebido)
🎁 Bônus: (bônus com valor)

De ~~R$ (preço original)~~ por apenas **R$ (preço atual)**
(ou parcelas de R$ (valor parcela))

🛡️ Garantia de (prazo) dias

[BOTÃO: (CTA — ex: QUERO MINHA VAGA AGORA)]
```

#### Genuine Urgency
```
⚠️ (Razão real para urgência — ex: "Turma limitada a 30 alunos para garantir acompanhamento individual")

(Dado que comprova — ex: "Na última turma, as vagas esgotaram em 48 horas")

Não estou dizendo isso pra te pressionar. Estou dizendo porque (motivo genuíno).

[BOTÃO: (CTA)]
```

#### Two Options Template
```
Agora você tem uma escolha:

**Opção A:** Fechar essa página, voltar pra rotina de (dor), e continuar (consequência de não agir). Daqui a (prazo), nada terá mudado.

**Opção B:** Clicar no botão abaixo, investir em (transformação), e em (prazo) estar (resultado positivo específico).

Qual versão de você vai existir daqui a (prazo)?

[BOTÃO: (CTA — ex: ESCOLHO A TRANSFORMAÇÃO)]
```

#### Emotional Close Template
```
Eu sei o que é estar onde você está agora.

(Frase de empatia com a situação atual do leitor).

Mas eu também sei o que está do outro lado. Porque eu já estive aí — e (história breve de transformação pessoal ou de alunos).

Você merece (resultado desejado). E está a um clique de distância.

[BOTÃO: (CTA)]
```

#### Recap + CTA Template
```
Vamos recapitular tudo o que você leva hoje:

📦 (Item principal — valor R$ X)
📦 (Item 2 — valor R$ X)
📦 (Item 3 — valor R$ X)
🎁 Bônus 1: (nome — valor R$ X)
🎁 Bônus 2: (nome — valor R$ X)

**Valor total: R$ (soma)**
**Seu investimento hoje: R$ (preço real)**

🛡️ Garantia (incondicional/condicional) de (prazo) dias

[BOTÃO: (CTA)]

(Frase final de reforço — ex: "Se não for agora, quando?")
```

## WHEN TO USE ACCA

<!-- SYSTEM: Choose ACCA over AIDA when the audience is sophisticated, skeptical, or the offer requires education before desire. ACCA excels for complex/high-ticket offers where the buyer needs to understand WHY it works before they want it. -->

**Best for:**
- Audiências sofisticadas que já viram muitas promessas
- Ofertas complexas que precisam de explicação
- High-ticket onde a decisão é racional + emocional
- Mercados céticos ou saturados
- Produtos com diferencial técnico/metodológico

**Not ideal for:**
- Ofertas de impulso / low-ticket
- Públicos frios que precisam de gancho emocional rápido (use AIDA)
- Produtos simples e auto-explicativos

## ACCA vs AIDA Comparison

| Aspecto | ACCA | AIDA |
|---------|------|------|
| Foco | Educação → Convicção | Emoção → Desejo |
| Público ideal | Sofisticado, cético | Frio, emocional |
| Ticket | Médio-alto | Qualquer |
| Comprimento | Mais longo (precisa educar) | Pode ser curto |
| Prova | Central (Conviction) | Suporte (Desire) |
| CTA | Lógico + emocional | Emocional + urgência |
| Melhor para | Mentorias, consultorias, SaaS | Cursos, e-books, eventos |
