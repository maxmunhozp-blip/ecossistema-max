# Russell Brunson — Key Frameworks for Sales Pages

> Reference file for sales page copy generation. Claude should apply these frameworks
> when building story sections, offer stacks, and belief-breaking sequences.

---

## 1. Epiphany Bridge

The core storytelling framework. Every transformation story on a sales page should follow this structure.

### Structure (7 beats):

1. **Backstory** — Quem era você ANTES. Contexto, situação, rotina.
2. **Wall** — O obstáculo que apareceu. O momento de crise/frustração.
3. **Epiphany** — O insight que mudou tudo. O momento "eureka".
4. **Plan** — O que você fez a partir daquele insight.
5. **Result** — O resultado concreto que conquistou.
6. **Flaw** — A dificuldade/imperfeição no caminho (gera empatia).
7. **Moral** — A lição universal que o leitor pode aplicar.

### Template PT-BR:

```
[BACKSTORY]
Eu era [situação anterior]. Todos os dias eu [rotina/frustração].

[WALL]
Até que um dia [evento/crise]. Eu tentei [soluções que falharam] e nada funcionava.

[EPIPHANY]
Foi quando eu descobri que [insight]. Percebi que o problema nunca foi [crença antiga] — era [nova perspectiva].

[PLAN]
Então eu [ação tomada]. Criei [método/sistema] baseado em [princípio].

[RESULT]
Em [tempo], eu [resultado mensurável]. [Prova social/número].

[FLAW]
Não foi perfeito. [Dificuldade real]. Mas [como superou].

[MORAL]
E foi aí que eu entendi: [lição universal aplicável ao leitor].
```

## 2. The Stack / Value Stack

Build perceived value by stacking components visually and rhetorically. Each item gets its own "slide" with individual value.

### How to build:

1. Liste cada componente da oferta separadamente
2. Atribua um valor individual realista a cada um
3. Apresente um por um, acumulando o valor total
4. Mostre o valor total empilhado vs. o preço real
5. Feche com "Diga apenas talvez" (just say maybe)

### Slide format per item:

```
[COMPONENTE]: [Nome atraente]
Valor: R$ [valor individual]
[1 frase sobre o que resolve]
```

### Stack close template:

```
Recapitulando tudo que você recebe:
✓ [Item 1] — Valor: R$ X
✓ [Item 2] — Valor: R$ X
✓ [Item 3] — Valor: R$ X
✓ [Bônus 1] — Valor: R$ X
✓ [Bônus 2] — Valor: R$ X

Valor total: R$ [soma]
Seu investimento hoje: R$ [preço real]
```

### "Just Say Maybe" close:

```
Você não precisa decidir agora. Diga apenas "talvez".
Teste por [prazo da garantia] dias. Se não for tudo que prometi — e mais — você recebe cada centavo de volta.
```

## 3. Trial Close Technique

Micro-commitments throughout the page that get mental "yes" before the actual CTA.

### Application:

Insira perguntas retóricas ao longo da página que o leitor responde "sim" mentalmente:

- "Faz sentido pra você?" (após explicar o método)
- "Você consegue se ver fazendo isso?" (após mostrar resultado)
- "Se isso funcionasse pra você, valeria a pena?" (antes do preço)
- "Imagina como seria sua vida se [resultado]?" (seção de transformação)

### Rule: Use 3-5 trial closes before the main CTA.

## 4. Two Journeys of the Hero

Every offer promises TWO transformations. The sales page must address both.

| Journey | Description | Page Section |
|---------|-------------|--------------|
| **External (Achievement)** | O resultado tangível, mensurável | Bullets de entrega, números, provas |
| **Internal (Transformation)** | Quem a pessoa se TORNA | História, identidade, visão de futuro |

### Template:

```
[EXTERNAL]
Com [produto], você vai [resultado concreto] em [prazo].

[INTERNAL]
Mas o mais importante: você vai se tornar [nova identidade].
Vai acordar sabendo que [sentimento/estado]. Vai olhar no espelho e ver [transformação interna].
```

## 5. Big Domino Theory

Identify the ONE core belief that, if knocked down, makes everything else fall into place. The entire page should be structured to topple this single domino.

### Formula:

> "Se eu consigo fazer eles ACREDITAREM que [nova oportunidade] é a chave para [resultado desejado] e que só é possível através do [meu veículo específico], então todas as outras objeções se tornam irrelevantes."

### Application:

1. Identifique a crença central limitante do público
2. Estruture toda a página para derrubar ESSA crença
3. Cada seção (história, prova, oferta) reforça a quebra dessa crença

---

## 6. New Opportunity vs. Improvement

ALWAYS position the offer as a NEW OPPORTUNITY, never as an improvement on what they're already doing.

### Why: People don't want to improve something they associate with failure. They want something NEW.

| Improvement (AVOID) | New Opportunity (USE) |
|---------------------|----------------------|
| "Melhore suas vendas" | "Um novo sistema de vendas que..." |
| "Aprenda a vender mais" | "Descubra o método que substitui..." |
| "Otimize seus anúncios" | "A nova estratégia que torna anúncios obsoletos" |
| "Emagreça com dieta" | "O protocolo que elimina a necessidade de dieta" |

## 7. 3 Secrets Framework

Three core beliefs to break on every sales page:

| Secret | Belief Type | What to Break |
|--------|------------|---------------|
| **Secret #1: The Vehicle** | "Esse método funciona?" | Prove que o veículo (método/produto) é válido |
| **Secret #2: Internal** | "EU consigo fazer isso?" | Quebre a crença de que eles não são capazes |
| **Secret #3: External** | "Vai funcionar PRA MIM?" | Elimine objeções externas (tempo, dinheiro, situação) |

### Page mapping:

- **Secret 1** → Seção de método/mecanismo + provas
- **Secret 2** → Seção de história + depoimentos de pessoas "normais"
- **Secret 3** → Seção de FAQ + garantia + suporte

## 8. Perfect Webinar Structure (Adapted for Sales Pages)

The webinar flow translated into page sections:

| Webinar Segment | Sales Page Section |
|-----------------|-------------------|
| Big Promise | Hero / Headline |
| Ruler (qualification) | "Para quem é" section |
| 3 Secrets | Body sections (método + provas) |
| The Stack | Oferta detalhada |
| Closes / Urgency | CTA + escassez + garantia |
| Q&A | FAQ section |

## 9. Dream 100 (for Authority Section)

Use Dream 100 thinking to build the authority section:

### Application:

- Mencione associações com autoridades reconhecidas
- Liste mídia/publicações onde apareceu
- Mostre logos de empresas/clientes relevantes
- Cite números de audiência/comunidade
- Use "Como visto em..." / "Confiado por..."

### Template:

```
[AUTORIDADE]
Mais de [número] [alunos/clientes/empresas] já [resultado].
Como visto em [mídia/publicação].
[Mentor/parceiro] de [nome reconhecido].
```
