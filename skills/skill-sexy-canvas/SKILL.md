# SEXY CANVAS — Copy Engine para Trafego Direto

## O Framework

O SEXY CANVAS mapeia 8 gatilhos emocionais primitivos para gerar copies de alta conversao.
Baseado nos 7 pecados capitais + crianca interior, com 7 motivadores base de suporte.

### Os 8 Gatilhos (Pecados Capitais + Crianca Interior)

| # | Gatilho | Desejo Primitivo | Pergunta-Chave |
|---|---------|-----------------|----------------|
| 1 | **GANANCIA** | Ter mais, acumular, vantagem | "Como a pessoa GANHA mais do que pagou?" |
| 2 | **LUXURIA** | Desejo intenso, prazer, atracao | "O que torna isso IRRESISTIVEL?" |
| 3 | **INVEJA** | Querer o que o outro tem | "O que os OUTROS vao pensar quando virem?" |
| 4 | **IRA** | Raiva, frustracao, indignacao | "Qual INJUSTICA revolta essa pessoa?" |
| 5 | **CRIANCA INTERIOR** | Nostalgia, diversao, leveza | "Como isso resgata ALEGRIA e simplicidade?" |
| 6 | **GULA** | Querer tudo, excesso, abundancia | "Como fazer a pessoa querer MAIS e MAIS?" |
| 7 | **PREGUICA** | Facilidade, atalho, conveniencia | "Como isso SIMPLIFICA a vida drasticamente?" |
| 8 | **VAIDADE** | Ego, status, superioridade | "Como isso faz a pessoa se sentir ESPECIAL?" |

### Os 7 Motivadores Base (Sustentam os Gatilhos)

| Motivador | Funcao | Gatilhos que Amplifica |
|-----------|--------|----------------------|
| **Curiosidade** | Abre o loop mental | Ganancia, Inveja, Luxuria |
| **Recompensa** | Promete retorno | Ganancia, Gula |
| **Diversao** | Gera engajamento | Crianca Interior, Luxuria |
| **Seguranca** | Remove medo | Preguica, Ira |
| **Liberdade** | Promete autonomia | Preguica, Vaidade, Ira |
| **Amor** | Conecta emocionalmente | Crianca Interior, Inveja |
| **Pertencimento** | Cria tribo | Vaidade, Inveja, Crianca Interior |

## Processo de Execucao

### FASE 1 — Coleta de Contexto

Coletar do usuario (ou do product-marketing-context se existir):

```
PERSONA:
- Quem e? (genero, idade, profissao, momento de vida)
- Qual a dor #1? (frustracao central)
- Qual o desejo #1? (sonho/aspiracao)
- O que ja tentou? (solucoes anteriores que falharam)

EXPERIENCIA/OFERTA:
- Nome do produto/servico
- Preco (ou faixa)
- Promessa principal (transformacao)
- Diferencial (por que ESSE e nao outro)
- Prova (numeros, depoimentos, resultados)

TRAFEGO:
- Plataforma (Meta, Google, TikTok, YouTube)
- Temperatura (frio, morno, quente)
- Formato (imagem, video/reels, carrossel)
- Destino (LP vendas, checkout, whatsapp, webinar)
```

### FASE 7 — Compliance e Seguranca

#### Gatilhos que Exigem Cuidado (Politicas Meta/Google/TikTok)

| Gatilho | Risco | Como Mitigar |
|---------|-------|-------------|
| **IRA** | Conteudo negativo, linguagem agressiva | Redirecionar ira pra SISTEMA, nunca pra PESSOA. Evitar "voce esta errado" |
| **INVEJA** | Comparacao direta, bullying | Usar "inspiracao" nao "comparacao". Nunca "voce e pior que" |
| **GANANCIA** | Promessas financeiras, renda | NUNCA prometer valores especificos sem disclaimer. Usar "resultados variam" |
| **LUXURIA** | Conteudo sensual/apelativo | Manter no desejo EMOCIONAL, nao fisico. Foco em experiencia |
| **GULA** | Exagero, claims irreais | Basear em numeros reais. "Mais de 200 aulas" e OK se for verdade |
| **VAIDADE** | Elitismo, exclusao | Usar "exclusivo" com cuidado. Nao dizer "pra quem nao pode nao e" |

#### Palavras/Frases PROIBIDAS (Auto-Reprovacao)
- "Fique rico rapido" / "Enriqueca" / "Dinheiro facil"
- "Garantido" (resultado financeiro)
- "Cure" / "Trate" / "Elimine" (saude)
- "100% comprovado" / "Cientificamente provado" (sem fonte)
- Mencao a atributos pessoais: "Voce que e gordo/magro/feio/pobre"
- Clickbait enganoso: "Voce nao vai acreditar"
- Claims absolutos: "O unico", "O melhor do mundo", "Nunca visto"
