# Skill: Criador de Apresentação

Gera estrutura completa de apresentações para diferentes objetivos: pitch de produto, proposta comercial, aula ou keynote.

## TIPOS DE APRESENTAÇÃO

### TIPO 1 — PITCH DECK (produto/startup) — 12 slides
1. **Capa** — Nome + tagline impactante
2. **Problema** — A dor que você resolve (com dado/estatística)
3. **Solução** — O que você faz em 1 frase
4. **Demo / Como funciona** — 3 telas ou 3 passos
5. **Mercado** — Tamanho da oportunidade (TAM/SAM/SOM simplificado)
6. **Tração** — O que já aconteceu (usuários, receita, crescimento)
7. **Modelo de negócio** — Como você ganha dinheiro
8. **Diferenciais** — Por que você vence a concorrência
9. **Time** — Quem está construindo e por que é o time certo
10. **Roadmap** — Próximos 3 marcos
11. **Pedido** — O que você precisa (investimento, parceria, cliente)
12. **Próximos passos** — CTA específico

### TIPO 2 — PROPOSTA COMERCIAL — 8 slides
1. **Capa personalizada** — Logo do cliente + "Proposta para [Nome]"
2. **Entendemos seu desafio** — Resume o problema deles com as próprias palavras
3. **Nossa solução para você** — Específica, não genérica
4. **Como vamos trabalhar** — Processo em 3-4 fases
5. **O que você vai ter** — Entregáveis claros com prazo
6. **Investimento** — Preço + forma de pagamento + o que inclui
7. **Por que nós** — 3 razões + prova social do mesmo nicho
8. **Próximos passos** — O que acontece depois de aprovar

### TIPO 3 — AULA / TREINAMENTO — 10 slides
1. **Capa** — Título + promessa de transformação
2. **Agenda** — O que vamos ver hoje (antecipa valor)
3. **Quem sou eu** — Credencial relevante para o tema
4. **O problema** — Por que isso importa para eles
5. **Conceito central** — A grande ideia em 1 frase
6. **Parte 1** — Primeiro bloco de conteúdo
7. **Parte 2** — Segundo bloco de conteúdo
8. **Parte 3** — Terceiro bloco de conteúdo
9. **Resumo + Aplicação** — O que fazer agora (tarefa prática)
10. **Próximos passos** — Como continuar aprendendo

## COMO USAR

Diga: **"cria [tipo] de apresentação sobre/para [tema/produto/cliente]"**

Exemplos:
- "cria pitch deck do TractorPRO"
- "apresentação comercial para vender o TractorPRO para agências"
- "estrutura de aula sobre tráfego pago para iniciantes"
- "deck de vendas do meu produto de gestão"

## ENTREGÁVEIS

A skill gera:
1. Roteiro completo slide a slide
2. Copy de cada slide (título + 3-5 bullets ou parágrafo)
3. Notas do apresentador (o que falar em cada slide)
4. Sugestões de design (cores, imagens, ícones para cada slide)
5. Tempo estimado de apresentação
