Voce tem um computador novo (ou formatado) e quer rodar tudo do mesmo jeito de antes.

1

Instalar dependencias

No novo Mac/Linux, instalar Node, Git, gh CLI, e Claude Code:

brew install node git gh
# Claude Code
curl -fsSL https://claude.ai/install.sh | sh
2

Login no GitHub

gh auth login

Escolhe HTTPS, autentica com seu navegador.

3

Clonar os 3 repos privados

cd ~
git clone https://github.com/maxmunhozp-blip/claude-backup.git
git clone https://github.com/maxmunhozp-blip/ecossistema-max.git
mkdir -p ~/Documents
git clone https://github.com/maxmunhozp-blip/esmeralda-vault.git ~/Documents/Obsidian\ Vault
4

Restaurar memoria + skills + settings do Claude

Ajuste o nome do user (no exemplo "maxmunhoz") pro seu nome de usuario real:

USER_DIR="-Users-$(whoami)"
mkdir -p ~/.claude/projects/$USER_DIR
cp -r ~/claude-backup/memory ~/.claude/projects/$USER_DIR/
cp -r ~/claude-backup/skills/* ~/.claude/skills/ 2>/dev/null || mkdir -p ~/.claude/skills && cp -r ~/claude-backup/skills/* ~/.claude/skills/
cp ~/claude-backup/settings.json ~/.claude/
cp ~/claude-backup/settings.local.json ~/.claude/
5

Login no Claude

claude

Ele vai pedir login. Faz com sua conta do Claude.ai (qualquer uma — Pro, Team, novo email — funciona). A memoria local ja vai estar la.

6

Instalar Obsidian + plugin Git

Baixa Obsidian em obsidian.md, abre o vault em ~/Documents/Obsidian Vault, ativa community plugins, instala "Obsidian Git", configura com o token do CREDENCIAIS.md.

7

Pronto

Abre Claude no projeto: cd ~/ecossistema-max && claude

A IA ja vai saber tudo sobre voce, seus clientes, projetos, sociedades. Pode continuar de onde parou.
